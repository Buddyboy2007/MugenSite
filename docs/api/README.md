# API Documentation

This directory contains API documentation for the MugenSite project.

## Available APIs

- [Character API](./character-api.md) - Character data and management
- [Fighter API](./fighter-api.md) - Fighter information and statistics
- [User API](./user-api.md) - User management and authentication

## API Standards

- RESTful design patterns
- JSON responses
- Error handling with appropriate HTTP status codes
- Rate limiting and security measures
